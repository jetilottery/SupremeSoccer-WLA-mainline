Golazo Millonario

¡Goool! ¡Ponte los botines y entra a la cancha en Golazo Millonario! Revela los valores de los premios debajo de las camisetas en cada fila.

Si consigues 3 valores de premios idénticos en la misma fila, ¡ganaste! Junta las pelotas en los valores de premios para incrementar el medidor de goles y activar la ronda de bonificación. ¡Mantén la sangre fría en los penales mientras revelas a los pateadores para intentar ganar uno de los cinco niveles de premios!

Cómo apostar

Costo del boleto

Muestra el costo actual del boleto.

Presiona la flecha izquierda (-) para reducir el costo del boleto.

Presiona la flecha derecha (+) para aumentar el costo del boleto.

Presiona COMPRAR para descontar el costo del boleto de tu saldo y jugar.

Mostrar todos

Presiona MOSTRAR TODOS para completar el juego automáticamente. El juego continúa hasta que se revelen todos los símbolos.

Para continuar el juego de forma manual, presiona DETENER.

Cómo jugar

El objetivo de Golazo Millonario es encontrar tres valores de premios idénticos en una misma fila o juntar 3 pelotas de fútbol para activar la ronda de bonificación.

Selecciona cada una de las camisetas para revelar un valor de premio. Encuentra 3 valores de premio idénticos en una fila para ganar.

Los valores de premio también contienen 0, 1 o 2 pelotas de fútbol. Si revelas 3 pelotas se te otorga una ronda de bonificación.

En la ronda de bonificación, elige entre 10 objetivos para revelar jugadores de fútbol o X. Cada jugador que reveles aumenta el valor de la copa del premio de bonificación instantánea. Cuando reveles 3 X, la bonificación termina, y los premios obtenidos se agregan al premio total.

Para completar el juego, elige todas las camisetas o presiona MOSTRAR TODOS, y aparecerá un mensaje que indicará el resultado del juego, y este finalizará.

Reglas

El uso indebido o mal funcionamiento anulan todos los pagos y jugadas.

Los premios de la bonificación se agregan al valor de premio total.

Los premios se muestran como monedas.

Las decisiones del jugador durante la partida no afectan el resultado del juego.
