Supreme Soccer

Vous gagnez lorsque trois lots identiques apparaissent sur une rangée! Comment jouer

L’objectif du jeu Supreme Soccer est de trouver trois lots identiques dans une rangée ou d’accumuler trois ballons de soccer afin de déclencher le jeu boni.

Choisissez des chandails pour faire apparaître le lot qu’il cache. Vous gagnez lorsque trois lots identiques apparaissent sur une rangée.

Les affichettes de lots contiennent aussi zéro, un ou deux ballons de soccer. Trouvez trois ballons de soccer pour déclencher le jeu boni.

Dans le jeu boni, choisissez parmi les dix cibles pour faire apparaître des joueurs de soccer ou des « X ». Chaque joueur trouvé augmente la valeur du gain instantané comme indiqué sur la coupe. Le jeu boni prend fin lorsque vous avez trouvé trois « X », et le lot affiché sur la coupe, le cas échéant, est ajouté à votre gain total.

Pour compléter le jeu, sélectionnez tous les chandails ou cliquez sur {REVEALALL}, ce qui fait apparaître un message indiquant le résultat de la partie, et la partie prend fin.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les lots instantanés sont ajoutés aux gains totaux.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
